# yumii

## My personal discord music bot :)



### Menu 🎶🎵
### To pull up the menu
-help

### Play/add a song to music queue ▶
-p *song name*
### Check now playing ❓
-np
### Check music queue ⌛
-q
### Skip to next song ⏩
-skip
### Set loop to NONE, CURRENT or PLAYLIST 🔁
-loop
### Pause the track ⏸
-pause
### Resume the track ⏯
-resume
### Connect to voice channel 🚩
-connect
### Disconnect from voice channel ⛔
-disconnect
